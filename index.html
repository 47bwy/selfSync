本网站制作和写作方式

使用 mkdocs 和 markdown 构建，使用 Python-Markdown-Math 完成数学公式。

安装依赖：

requirements.txt

使用 pipenv 安装

编写并查看：

mkdocs serve    # 修改自动刷新浏览器，浏览器打开 http://localhost:8000 访问
mkdocs gh-deploy    # 部署到自己的 github pages, 如果是 readthedocs 会自动触发构建
